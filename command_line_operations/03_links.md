# Linux Command Line – Links

| Command | Purpose                         | Example                     |
| ------- | ------------------------------- | --------------------------- |
| `ln`    | Create **hard link**             | `ln file1 file2`            |
| `ln -s` | Create **symbolic (soft) link** | `ln -s file1 link_to_file1` |
