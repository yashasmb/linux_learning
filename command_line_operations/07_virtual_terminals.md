# Linux Command Line – Virtual Terminals

* **VT** → multiple text consoles outside GUI  
* Switch between text VTs: **Ctrl+Alt+F1–F6**  
* Switch back to GUI: **Ctrl+Alt+F7** (or F1 on some systems)
